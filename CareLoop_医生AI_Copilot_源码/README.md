# CareLoop 医生 AI Copilot

一套贯穿诊前、诊中、诊后连续照护闭环的高保真交互原型。产品以医生工作台为核心，把患者资料整理、风险队列、证据溯源、照护计划审核与患者任务执行串成一条可演示的完整路径。

> 重要：本仓库仅用于产品原型与方案验证，所有患者、指标和诊疗信息均为虚构演示数据，不可用于真实诊疗。

## 已实现的体验

- 医生工作台：风险分层、患者搜索、待办队列、今日接诊和 AI 晨间简报
- 患者纵向档案：诊前摘要、来源证据、指标趋势、用药与任务概览
- 照护计划审核：目标、患者任务、风险规则、患者友好说明和医生发布
- 患者端：今日任务、血压录入、症状反馈、求助入口和完成度更新
- 安全设计：AI 内容待人工审核、来源可追溯、演示数据标识、医生最终决策

## 演示路径

1. 在工作台打开高风险患者“李建国”。
2. 在患者档案查看 AI 诊前摘要及 8 条引用来源。
3. 进入照护计划，审核任务并点击“确认并发布计划”。
4. 打开患者端，录入血压并观察任务进度从 1/3 更新为 2/3。

## 技术栈

- React 19
- Next.js 16 / Vinext
- TypeScript
- Cloudflare Workers 兼容构建
- 原生 CSS，无额外 UI 组件依赖

## 本地运行

需要 Node.js `>=22.13.0`。

```bash
npm ci
npm run dev
```

浏览终端输出的本地地址即可。生产构建与校验：

```bash
npm run build
npm run validate:artifact
```

## 下一阶段建议

当前版本是前端高保真原型。进入真实 MVP 开发时，建议依次接入身份与权限、患者主索引、结构化病历接口、审计日志、任务编排、消息通知、规则引擎和经过临床验证的 AI 服务，并完成安全、合规与临床有效性评审。
